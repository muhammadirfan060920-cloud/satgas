# 🛡️ SatGas AI v2 (SADIS Edition) - Sangat Diplomatis & Solutif
**"Karena Hidup Terlalu Berharga untuk Habis di Slot dan Pinjol"**

SatGas AI v2 adalah prototipe asisten otonom tingkat lanjut yang dirancang khusus untuk melindungi masyarakat Indonesia dari jeratan ekosistem Judi Online (Judol) dan Pinjaman Online (Pinjol) Ilegal. Dibangun menggunakan **Gemini 3 Flash**, aplikasi ini melampaui sekadar chatbot biasa dengan kemampuan analisis otonom dan proteksi privasi tingkat tinggi.

---

## 🚀 Fitur Utama

### 1. 📂 Multimodal Financial Analyzer
Agent dapat menganalisis data finansial langsung dari screenshot (mutasi bank, bill pinjol, dll). Cukup unggah foto, dan Agent akan mengekstrak informasi penting secara otonom untuk dianalisis risikonya.

### 2. 🚨 Crisis Mode & Sentinel Detection
Sistem secara proaktif mendeteksi tanda-tanda krisis mental atau niat menyakiti diri sendiri dalam percakapan. Jika terdeteksi, Mode Krisis diaktifkan secara otomatis, menampilkan banner darurat dan akses cepat ke **Hotline 119**.

### 3. 📝 Autonomous Negotiation Strategy
Bukan sekadar *template* surat. Agent secara otonom **memilih strategi negosiasi terbaik** berdasarkan profil risiko Anda:
*   **High Risk/Crisis**: Memprioritaskan *Penghapusan Bunga & Denda*.
*   **Moderate Risk**: Memprioritaskan *Restrukturisasi Tenor* (Perpanjangan Waktu).
*   Semua draf mengacu pada **POJK Nomor 11/POJK.03/2020**.

### 4. 🔍 OJK Validator & "Ghost Protocol"
*   **Auto-Scan & Manual**: Memvalidasi legalitas pinjol terhadap simulasi database OJK.
*   **Ghost Protocol**: Jika terdeteksi **Pinjol Ilegal**, Agent secara otonom akan memerintahkan protokol penghentian pembayaran, penggantian nomor, dan pelaporan ke Satgas PASTI (sesuai arahan otoritas).

### 5. 🔔 Budgeting Sentinel (Privacy-First)
Sistem pemantauan pengeluaran otonom dengan alert yang berjalan 100% di sisi client.
*   **Toaster Alerts**: Notifikasi otonom tanpa memerlukan data PII (Email/WA).
*   **Debt-to-Income (DTI) Chart**: Visualisasi *shock therapy* rasion hutang yang berubah warna menjadi merah saat melewati batas aman 30%.

### 6. 🏆 Quest & Milestones
Sistem gamifikasi untuk mendukung perjalanan pemulihan pengguna. Melacak hari bersih dari judi dan memberikan badge serta medali virtual untuk mencapai target tertentu.

## 🛠 Tech Stack

- **Framework**: [React 19](https://react.dev/) + [Vite](https://vitejs.dev/)
- **Styling**: [Tailwind CSS v4](https://tailwindcss.com/)
- **AI Core**: [@google/genai](https://www.npmjs.com/package/@google/genai) (Gemini 3 Flash)
- **Animations**: [Framer Motion](https://www.framer.com/motion/)
- **Charts**: [Recharts](https://recharts.org/)
- **PWA**: [vite-plugin-pwa](https://vite-pwa-org.netlify.app/)
- **Offline Capabilities**: Menggunakan sistem *Pre-cached Instructions*, Agent tetap dapat memberikan panduan darurat dasar bahkan saat koneksi internet terputus.
- **UI Components**: [Lucide React](https://lucide.dev/), [Sonner (Toasts)](https://sonner.emilkowal.ski/)

---

## 🛡️ Filosofi Privasi
Aplikasi ini dirancang dengan prinsip **Anonymous by Design**:
- Tidak memerlukan login.
- Analisis data dilakukan secara lokal/tanpa database permanen untuk identitas pengguna.
- Tidak menyimpan data PII (Personally Identifiable Information).
- Notifikasi bersifat lokal di perangkat pengguna.

---

## 🇮🇩 Konteks Indonesia
SatGas AI dikembangkan dengan mempertimbangkan lanskap finansial unik di Indonesia:
- Deteksi istilah slang lokal terkait judi online dan pinjol.
- Integrasi sumber daya bantuan hukum dan psikologis lokal (LBH, Safeguard).
- Kepatuhan simbolis terhadap regulasi OJK dalam draf hukum.
